Purpose
-------------------

* The web app reports simple metrics to record performance of the app and to better understand what is used and how.
* The metrics recorded are very currently very minimal and are completely anonymous.
* Major app updates may reset opt-out status. Minor app updates will not reset out-out status.

What is reported
-------------------

* Site visits
* Labels printed
* Label type
* Time between page load (or time since last label printed) to actual label print
* Time/date printed
* Session ID (timestamp of when page was loaded)

What isn't reported
-------------------

* Name
* Computer
* Department
* Any label data
* Anything that is not listed in the above section.
* This is subject to change at any point in time, and we may add metrics for a better idea of how the app is being used.

How is this data used, and who sees it?
-------------------

* This data will only be used by the maintainer/developer to gather some insight on label usage and how the app is used.
* Generalized data may be shown to management if needed. As stated before, data cannot be tied back to one individual person.

Opting in/out
-------------------

* By default, metrics are enabled.
* You may opt out of these metrics by [clicking here](https://fwc-sgomez.github.io/label-gen?disableMetrics=true)
* If you have opted out, but would like to opt back in, [click here](https://fwc-sgomez.github.io/label-gen?disableMetrics=false)
